Font Icons for .NET MAUI

mauicons is a font that contains icons from GitHub Octicons and Ionicons, both of which are available in the MIT License. The icons are adjusted in size and combined into a single font of 884 icons so that it can be easily used in the development of Open UI Kit for .NET MAUI (#MAOUI). The font is in True Type format and available under MIT License.
